# Sisteme_Inteligente_proiect

## DESCRIERE ##
Scopul acestui proiect este de a analiza obiceiuri din viața de zi cu zi a studenților. Pe baza acestor date, se poate „prezice” performanța pe care un student o va avea, măsurată atât prin **nota finală** obținută, cât și prin **scorul productivității**.

## DATASET ##
Sursa: [Student Productivity & Digital Distraction Dataset](https://www.kaggle.com/datasets/sehaj1104/student-productivity-and-digital-distraction-dataset)

Acesta cuprinde informații despre: 
- stilul de viață
- comportamentul academic
- distrageri digitale
- metrici de productivitate
